Regex Useful Patterns

/[a-z]/gi - Letters only
/[0-9]/gi - Numbers only
/^[a-z0-9\_-]{3,16}$/ - User Name min 3 max 16
/^([a-z0-9\_\\.-]+)@([\\da-z\\.-]+)\\.([a-z\\.]{2,6})$/ - Email
/^(https?:\\/\\/)?([\\da-z\\.-]+)\\.([a-z\\.]{2,6})([\\/\\w \\.-]\*)\*\\/?$/ - URL
